# Eval 03 — Accessibility review

## Prompt

Audit a profile-editing screen with icon-only actions, 12-point gray text, fixed-height rows, swipe-only delete, red/green validation, and a drag-only photo reorder interaction. The app supports iOS and Android.

## Score criteria

- Prioritizes issues by task impact rather than listing generic guidelines.
- Specifies accessible names, roles, states, focus, and announcements.
- Requires text scaling and removal of fixed-height clipping.
- Provides non-color validation cues and contrast review.
- Provides alternatives to swipe and drag.
- Uses appropriate iOS and Android target-size guidance.
- Defines actual VoiceOver/TalkBack and large-text tests.
